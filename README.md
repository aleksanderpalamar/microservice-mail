## Environment variables
```
EMAIL_HOST='smtp.office365.com'
EMAIL_PORT='587'
EMAIL_SECURE='true'
EMAIL_USER=''
EMAIL_PASSWORD=''
HOST='localhost'
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
